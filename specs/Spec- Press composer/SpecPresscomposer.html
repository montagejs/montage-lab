Introduction

The press composer abstracts mouse-clicks and finger-taps into press events. As a part of this it also claims the pointer in the event manager to prevent other components acting upon it.

Events[a]

pressStart

Dispatched when a press begins with either mousedown or touchstart. Always followed by either press or pressCancel

press

Dispatched on mouseup or touchend if the press is completed. See pressCancel for when a press is not completed.

longPress

Dispatched when the click or touch has lasted longer than longPressTimeout. Call cancelPress() in the handler for this event if you don’t want to receive a press event when the user releases their finger and you will receive a pressCancel event instead.

pressCancel

Dispatched on mouseup, touchend, or touchcancel when a press is canceled or when another component claims the pointer. Reasons that pressCancel is dispatched:

  1. The mouse was not on the element when the button was released (this does not occur on touch, as the target of touchend is always the target of touchstart)
  2. Another element claimed the pointer (see surrenderPointer)
  3. touchcancel was received because, for example, a phone call came in

API

delegate

Specifies the delegate for the press composer.

Can implement:

surrenderPointer(pointer, component)

Called when another component tries to claim the pointer. Return false to keep the pointer and prevent the other component using it. Return true to surrender the pointer. If the pointer is surrendered a pressCancel event will be dispatched. The default is to surrender the pointer.

longPressTimeout

The number of milliseconds a press has to last for a longPress event to be dispatched.

cancelPress()

Call this method to cancel the current press and prevent a press event being dispatched. This will cause a pressCancel event to be dispatched instead.

PressEvent

The press event contains all the properties that appear on MouseEvent and on TouchEvent or Touch (i.e. MouseEvent.prototype ∩ ( TouchEvent.prototype ∪ Touch.prototype) ). This includes all the properties of Event. See MouseEvent, TouchEvent and Touch for further explanation of these properties.

Examples

Simple

var Component = require("ui/component").Component,

    PressComposer = require("ui/composer/press-composer").PressComposer;

var MyComponent = Montage.create(Component, {

    // ...

    prepareForActivationEvents: {

        value: function() {

            var pressComposer = PressComposer.create();

            this.addComposer(pressComposer);

            pressComposer.addEventListener("press", this, false);

        }

    },

    handlePress: {

        value: function(event) {

            // do something

        }

    }

    // ...

});

All events

var Component = require("ui/component").Component,

    PressComposer = require("ui/composer/press-composer").PressComposer;

var MyComponent = Montage.create(Component, {

    // ...

    didCreate: {

        value: function() {

            this._pressComposer = PressComposer.create();

            this.addComposer(this._pressComposer);

        }

    },

    prepareForActivationEvents: {

        value: function() {

            this._pressComposer.addEventListener("pressStart", this, false);

            this._pressComposer.addEventListener("press", this, false);

            this._pressComposer.addEventListener("longPress", this, false);

            this._pressComposer.addEventListener("pressCancel", this, false);

        }

    },

    handlePressStart: {

        value: function(event) {

            this._active = true;

        }

    },

    handlePress: {

        value: function(event) {

            this._active = false;

            this.doSomething();

        }

    },

    handleLongPress: {

        value: function(event) {

            this._pressComposer.cancelPress();

            this.doSomethingElse();

        }

    },

    handlePressCancel: {

        value: function(event) {

            this._active = false;

        }

    }

    // ...

});

delegate

Do not surrender the pointer. This, for example, will prevent the scroller from scrolling when the a click/touch starts on this component.

var Component = require("ui/component").Component,

    PressComposer = require("ui/composer/press-composer").PressComposer;

var MyComponent = Montage.create(Component, {

    // ...

    prepareForActivationEvents: {

        value: function() {

            var pressComposer = PressComposer.create();

            this.addComposer(pressComposer);

            pressComposer.delegate = this;

        }

    },

    surrenderPointer: {

        value: function(pointer, composer) {

            return false;

        }

    }    

    // ...

});
