ToggleButton Component

ToggleButton extends the Button component to add a “pressed” state. Each action on the button will toggle to the other state.

It supports all (most?) standard attributes of the <input> HTMLElement as specified here - http://www.w3.org/TR/html5/the-input-element.html#the-input-element

and http://www.w3.org/TR/html5/the-button-element.html#the-button-element

https://developer.mozilla.org/en/HTML/Element/Input

Properties

pressed

Type: Boolean

Default: false

Whether the toggle button is currently in the pressed state or not. The toggle button’s label will be set to pressedLabel and unpressedLabel when this value is true and false respectively.

unpressedLabel

Type: String

Default: the toggle button’s label

The value label should take when the button is in the unpressed state.

pressedLabel

Type: String

Default: the toggle button’s label

The value label should take when the button is in the pressed state.

pressedClass

Type: String

Default: “pressed”

A class that should be added to the element when the button is in the pressed state. It will be removed again when the button is unpressed.

label

Type: string

Default: The value property of an <input>, or the first text child node of a <button>

Text to be displayed in this component. For an <input> element this property maps directly to the the value property. For an <button> element this property changes the first child text node. If a text node doesn’t exist it will be created.

If this is set equal to either unpressedLabel or pressedLabel then the pressed property of the button will be changed to the respective state.

converter

Type: Object instance

Default: null

Optional. If provided, the converter will be used to convert the label before the label is displayed.

// Standard HTML5 Attributes

autofocus

Data Type = Boolean, Default value = false

disabled

Data Type = boolean, Default value = false

Doesn’t prevent the value to be changed programmatically via the value property

form

Data Type = String, Default value = null

formaction

Data Type = String, Default value = null

formenctype

Data Type = String, Default value = null

formmethod

Data Type = String, Default value = null

formnovalidate

Data Type = String, Default value = null

formtarget

Data Type = String, Default value = null

name

Data Type = String, Default value = null

value

Data Type = String, Default value = null

For an <input> element this is the same as the label property. On a <button> element this is the value sent with the form when the button is clicked.

Methods

toggle()

Changes the state of the toggle button.

Events

action

Fired when the button is used (clicked, touched, focused and Enter pressed etc.)

Markup & Serialization

This button will toggle between “On” and “Off”, starting in the pressed, “On” state.

<button class="btn" id="button1">Off</button>

"button1": {

   "module": "montage/ui/toggle-button.reel",

   "name": "ToggleButton",

   "properties": {

       "element": {"#": "button1"},

       "pressed": true,

       "pressedLabel": "On"

   },

   "listeners": [

       {

           "type": "action",

           "listener": {"@": "owner"}

       }

   ]

}
