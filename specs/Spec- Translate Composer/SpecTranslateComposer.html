Introduction

Translate composer aggregates mouse and touch events into motion calculations and applies after effects such as bounce and momentum to them.

Functional Description

Translate composer aggregates mouse and touch events into motion calculations and applies after effects such as bounce and momentum to them.  Components can use the translate-composer to implement motion based on input.

Events

translateStart

Dispatched when the translate composer is going to start motion calculations  

translateEnd

Dispatched when the translate composer is done calculating motion including any after effects such as bounce or momentum

translate

Dispatched for each change in the x or y direction, only dispatched if it’s being listened for

API

CustomEvent

type

translateStart | translateEnd | translate

translateX

Number: only available in events of type translate

translateY

Number: only available in events of type translate

Configurable Properties


pointerSpeedMultiplier[a]

Number: used to calculate the values for translateX and translateY based on input movement, default value is 1 which corresponds to a 1 to 1 translation between input movement and translate values, 2 would be a 1 to 2 translation between input movement and calculated translate values

maxTranslateX[b]

Number: the max value in the horizontal direction for input

maxTranslateY

Number: the max value in the vertical direction for input

axis

vertical | horizontal | both: the directions in which translations can be applied

invertAxis

Boolean: if true translate values are negative instead of positive, useful when the translate-composer is being used [c]for scrolling as opposed to an input for a control

hasMomentum

Boolean: whether or not momentum calculations should be performed based on the input

hasBouncing

Boolean: whether or not bouncing calculations should be performed based on the input

Calculated Properties

translateX

Number: the change in horizontal position calculated from the input

translateY

Number: the change in vertical position calculated from the input
