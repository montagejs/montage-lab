Introduction

The long press composer abstracts mouse click and hold and finger tap and hold events into a long press event.  Developers should use the long press composer in order to maintain consistency between the long press events in their applications.

Events

longpress

 Dispatched to indicate that a long press has occurred.  A long press is a mousedown or touchstart held for 1500ms with no movement beyond a threshold of 10px or a mouseup or touchend.

Examples

var Component = require("montage/ui/component").Component,

        LongPressComposer = require("montage/ui/composer/long-press-composer").LongPressComposer;

var MyComponent = Montage.create(Component, {

   // ...

   prepareForDraw: {

       value: function() {

           var longPressComposer = LongPressComposer.create();

           this.addComposer(longPressComposer);

           longPressComposer.addEventListener("longpress", this, false);

       }

   },

   handleLongpress: {

       value: function(event) {

           // do something

       }

   }

   // ...

});
