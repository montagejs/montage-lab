Button Component

Button is a Component wrapper for the <input type=”button”> and <button> HTMLElements. The HTML markup for the Button is the same as the standard HTML5 markup. Wrapping the <input> and <button> HTMLElements as a Montage Button component adds Data Binding support for all writable attributes of this element.

It supports all (most?) standard attributes of the <input> HTMLElement as specified here - http://www.w3.org/TR/html5/the-input-element.html#the-input-element

and http://www.w3.org/TR/html5/the-button-element.html#the-button-element

https://developer.mozilla.org/en/HTML/Element/Input

Properties

label

Type: string

Default: The value property of an <input>, or the first text child node of a <button>

Text to be displayed in this component. For an <input> element this property maps directly to the the value property. For an <button> element this property changes the first child text node. If a text node doesn’t exist it will be created.

converter

Type: Object instance

Default: null

Optional. If provided, the converter will be used to convert the label before the label is displayed.

// Standard HTML5 Attributes

autofocus

Data Type = Boolean, Default value = false

disabled

Data Type = boolean, Default value = false

Doesn’t prevent the value to be changed programmatically via the value property

form

Data Type = String, Default value = null

formaction

Data Type = String, Default value = null

formenctype

Data Type = String, Default value = null

formmethod

Data Type = String, Default value = null

formnovalidate

Data Type = String, Default value = null

formtarget

Data Type = String, Default value = null

name

Data Type = String, Default value = nullfin

type

Data Type = String, Default value = button

value

Data Type = String, Default value = null

For an <input> element this is the same as the label property. On a <button> element this is the value sent with the form when the button is clicked.

Events

action

Fired when the button is used (clicked, touched, focused and Enter pressed etc.)
