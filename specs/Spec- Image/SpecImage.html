Image Component

Image wraps a <img> HTMLElement and provides Data binding support. Image extends from NativeControl.

Properties

alt

Data Type = String, Default value = “”

height

Data Type = String, Default value = null

src

Data Type = String, Default value = null[a]

width

Data Type = String, Default value = null

title

Data Type = String, Default value = null
