Introduction

Due to Montage’s deferred draw loop a common pattern is to have accessors for a component property which do nothing except set a data property and then set needsDraw = true on the component with no other logic.  Therefore to save on code size and typing, it would be beneficial to add a new property descriptor, needsDraw, for data properties, which will implement this behavior.

Functional Description

Adding needsDraw: true to a data property descriptor will generate a getter and setter for the data property where the setter sets the value and sets needsDraw to true.  The getter will simply return the data property value.  If any custom logic is needed in the getter and setter for the data property a developer will not be able to use the needsDraw property descriptor.  Adding the needsDraw property descriptor to a function property or an accessor property is undefined and will do nothing.

API

Custom property descriptor

needsDraw

Boolean, set to true if an accessor should be generated which sets needsDraw = true when the value of the data property is set

Examples

var Montage = require("montage").Montage,

    Component = require("montage/ui/component").Component;

exports.Box = Montage.create(Component, {

    // Before

    _top: {

        value: 0

    },

    top: {

        get: function() {

            return this._top;

        },

        set: function(value) {

            this._top = value;

            this.needsDraw = true;

        }

    },

    // After

    _left: {

        value: 0,

        needsDraw: true

    },

});
