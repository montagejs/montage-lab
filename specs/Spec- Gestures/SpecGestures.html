Introduction

Montage should aggregate low level browser events into more granular custom events indicating gestures which application developers can have components in their applications listen for.

Functional Description

Montage should provide a system for creating gesture events for application developers and a set of predefined gestures that application developers can easily use in their application.  Some examples of these gesture events include a swipe event, a long press event and for devices with motion data a shake event.  The expectation is that more developers will be using pre-authored gesture recognizers as opposed to building their own.

API

To listen for these custom events a developer needs to do two things, first require the javascript file where the custom gesture is defined.  Second, add an event listener for the custom event which the gesture will fire. (See example.)

If a developer wants to author a new gesture they need to create a new gesture prototype created from the EventComposer prototype.  An important item to note when authoring new composed events is that the composer is a singleton.  State for individual cases of listeners cannot be maintained on the composer instance as there is only one instance of the composer for all cases that are listening.  The new gesture recognizer should implement two methods startObservingTarget_forEventType_(target, eventType) and stopObservingTarget_forEventType_(target, eventType). Event listeners for detecting the gesture should be added and removed in those methods respectively.  The new gesture prototype also needs to register itself with the default event manager using the event manager method registerComposer_forEventType_(composer, eventType).  (See example.)

long-press-composer.js

var Montage = require("montage").Montage,

    EventComposer = require("ui/event/composer/event-composer").EventComposer,

    defaultEventManager = require("core/event/event-manager").defaultEventManager;

var LongPressComposer = exports.LongPressComposer = Montage.create(EventComposer, {

startObservingTarget_forEventType_: {

        value: function (target, eventType) {

            if (window.Touch) {

                target.addEventListener("touchstart", this);

                target.addEventListener("touchmove", this);

                target.addEventListener("touchend", this);

            } else {

                target.addEventListener("mousedown", this);

            }

            ...

        }

},

stopObservingTarget_forEventType_: {

        value: function (target, eventType) {

            if (window.Touch) {

                target.removeEventListener("touchstart", this);

                target.removeEventListener("touchmove", this);

                target.removeEventListener("touchend", this);

            }

            else {

                target.removeEventListener("mousedown", this);

            }

            ...

        }

},

        ...

});

defaultEventManager.registerComposer_forEventType_(LongPressComposer.create(), "longpress");
