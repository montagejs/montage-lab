RadioButton[a] Component[b]

Radiobutton Component wraps a HTML <input type=”radio” ../> element and provides data binding support. Radiobutton extends from CheckInput.

Properties

checked

Data Type = boolean, Default value = false

Whether the checkbox is checked or not. Takes it’s value from the presence (true) or absence (false) of the checked attribute.
