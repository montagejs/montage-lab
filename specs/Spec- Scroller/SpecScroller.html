Introduction

The scroller provides a view into content that is bigger than itself. This content can be viewed by scrolling the scrollwheel or clicking and dragging a mouse, or by touching and dragging on touch devices. It provides several configuration options to control various aspects of the scrolling. It uses the translate-composer to provide much of its functionality. It is used by the List component.

API

scrollX

Type: number

Default: 0

The number of pixels that the content has been scrolled horizontally.

scrollY

Type: number

Default: 0

The number of pixels that the content has been scrolled vertically.

axis

Type: string, “vertical” | “horizontal” | “both”

Default: “both”

Which direction the scroller can be scrolled in.

displayScrollbars

Type: string, “vertical” | “horizontal” | “both” | “auto” | “none”

Default: “auto”

Which scrollbars to display. When set to “auto” the scroller will only display scrollbars if the content overflows in that direction.

hasMomentum

Type: boolean

Default: true

Whether the scroller should continue moving after the user has stopped dragging it.

momentumDuration

Type: number

Default: 650

The number of milliseconds the content should continue moving after the user releases the drag.

hasBouncing

Type: boolean

Default: true

Whether the scroller should allow the user to scroll beyond the end of the content, and then bounce it back into place once they release.

bouncingDuration

Type: number

Default: 750

The number of millisecond the content should take to bounce back into place.

Examples

Simple

{

    "scroller": {

        "module": "montage/ui/scroller.reel",

        "name": "Scroller",

        "properties": {

            "element": {"#": "scroller"}

        }

    }

}

<div data-montage-id="scroller" style="height: 300px">

    <p>Large paragraph...</p>

    <img src="the-universe.png" alt="The Universe">

    <p>Another large paragraph...</p>

</div>

Non-bouncy vertical scroller

{

    "scroller": {

        "module": "montage/ui/scroller.reel",

        "name": "Scroller",

        "properties": {

            "element": {"#": "scroller"},

            "hasBouncing": false,

            "hasMomentum": false,

            "axis": "vertical"

        }

    }

}

<div data-montage-id="scroller" style="height: 300px; width: 200px;">

    <ul>

        <li>I’m a list item</li>

        <li>Me too</li>

        <li>Me three</li>

        <li>And many more ...</li>

    <ul>

</div>
