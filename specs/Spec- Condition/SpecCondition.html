Condition Component

The condition component is a component which will hide or show content in an application based on its condition property.  There are two options for hiding content.  The first option is by the application and removal of a css class, montage-invisible, to the condition’s element.  The second option is by removal and insertion of the condition’s children elements into and out of the DOM.  The selection of the option to use can be made by setting the removalStrategy property.

Properties

condition

Boolean - the condition that is used to determine whether the condition’s content should be displayed or not

content

Component | Element - Accepts a component or element which will be hidden or shown based on the value of the component’s condition, defaults to the children elements of the condition’s element

removalStrategy

String, “remove” | “hide”, defaults to “remove” - Indicates the removal strategy for this condition, whether it should apply the CSS class montage-invisible, or manipulate the DOM.

Example

On a calculator display a “M” indication when there is a value stored in memory, or nothing if there is no value stored.  Achieve this by adding and removing the child element from the DOM.
