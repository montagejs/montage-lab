SelectInput[a] Component

Image wraps a <select> HTMLElement and provides Data binding support. SelectInput extends from NativeControl.

Properties

// Montage specific properties

contentController

Data Type = ArrayController, Default value = null

A contentController manages the items of the SelectInput. It also manages the selections.

If a contentController is not provided, a default contentController is provided by SelectInput and the content (<option>) of the Select element (from markup) is added to this. Eg: if the collection is provided via markup or using the “content” property, an ArrayController is created implicitly.

content

Data Type = Array, Default Value = null

The collection of items for the Select can optionally be provided via the “content” property. If the collection is provided in this way, the text and value property paths will typically be set. If not, “text” and “value” properties are assumed to be present in each item of the content Array.

valuePropertyPath

Data Type = String

If contentController is provided, this allows the developer to specify which property of the objects in the contentController  provides the "value" part of <option>

textPropertyPath

If contentController is provided, this allows the developer to specify which property of the objects in the contentController  provides the "text" attribute of <option>

Markup and Serialization

1) Content provided by the Markup. The contentController is not set explicitly by the Developer in this case. However, a default contentController is created automatically with the contents of the <select> element. The “selected” attribute of <option> is automatically set on the controller.

<select id="country" required>

                                <option value="">Please select a Country</option>

                                <option value="USA">United States of America</option>

                                <option value="INR" selected="true">India</option>

                                <option value="FR">France</option>

                                <option value="GER">Germany</option>

                            </select>

"country": {

            "module": "montage/ui/select-input.reel",

            "name": "SelectInput",

            "properties": {

                "element": {"#": "country"}

            }          

        },

5) Select an item within the collection

this.dept.contentController.selectedIndexes = [2];

Note that even though the contentController was not explicitly set, a contentController is implicitly created to maintain the SelectInput’s collection and manage selections.
